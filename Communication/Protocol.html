TYPE 1. TIME

TIME requests the current time of day, in hours. PAYLOAD is empty and SIZE 0.

It gets an RESPONSE with the followin RESPONSE-PAYLOAD:

HOUR

HOUR is one Byte and just a number between 0 and 23.

AUTH-ID is the Number with wich the system was unlocked and is a uint of size 4 Bytes.

The TIME is a field with SIZE 1 Byte and holds a number between 0-23, defining the hour the system was unlocked.

If this is sent to the Raspberry Pi that in turn will sent UNLOCK Command to the Servo Arduino.


TYPE 3: UNLOCK

Sent by the Node withe Raspberry PI ID to th e Node with the Servo ID that is then unlocking the system.

PAYLOAD is empty and SIZE 0.


TYPE 4: SHOW

Sents a String of ASCII Characters to be printed to the Screen.

PAYLOAD:

STRING

The STRING can be of variable SIZE and its

maximum length is the maximum of the SIZE Field.



TYPE 5: BEEP

If this Command is sent to the Output Node that Node will activate the Buzzer for a certain time.

PAYLOAD:

TIME

TIME is a 4 Byte number, it defines how many miliseconds the Buzzer shall be on.
